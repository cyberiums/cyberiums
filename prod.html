How does this work?

This Cyberium faucet is running on the Main network. To prevent malicious actors from exhausting all available funds or accumulating enough Cyberium to mount long running spam attacks, requests are tied to common 3rd party social network accounts. Anyone having a Twitter or Facebook account may request funds within the permitted limits.

To request funds via Twitter, make a tweet with your Cyberium address pasted into the contents (surrounding text doesn't matter).
Copy-paste the tweets URL into the above input box and fire away!

You can track the current pending requests below the input field to see how much you have to wait until your turn comes.

The faucet is running invisible reCaptcha protection against bots.
